# 电子书阅读器
## 技术栈
前端
- Vue: 2.6.11
- VueRouter: 3.2.0
- Vuex: 3.4.0

后端
- Express: 4.17.1
- Nginx: 1.19.6
- Node: 12.17.0
- MySql: 5.1.6
## 项目启动
```sh
npm install
npm run serve
```
